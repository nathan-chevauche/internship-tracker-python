# Internship Tracker

A simple CLI tool to manage and track job applications. This project focuses on clean OOP principles and basic data persistence.

### Features
- **OOP Architecture**: Modular design with `Company` and `Tracker` classes.
- **Persistence**: Data stored in `storage.json`.
- **Reliability**: Basic error handling for file I/O and data parsing.
- **Typing**: Python type hinting for better maintainability.

### Setup
```bash
# Clone the repo
git clone [https://github.com/nathan-chevauche/internship-tracker-python.git](https://github.com/nathan-chevauche/internship-tracker-python.git)
cd internship-tracker-python

# Environment setup
python -m venv .venv
source .venv/bin/activate  # Or .\.venv\Scripts\activate on Windows

# Run
python main.py
